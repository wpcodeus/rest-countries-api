==== Description ====

This is a software engineer technical assessment that utilizes the Rest Countries API as it's main data source. The main functionality consists or search for countries by name, full name, or code and view them as a list on a web page so that you can find the country data you're looking for.


== Installation ==

Download the zip file from Github and install it the files using your preferred file manager.


== Instructions ==

1. Enter country name or code into input field and hit 'Search'.
2. Browser the data to find the information about the country you're looking for.
3. Note: This project will only display the first 50 results.
